## 시작하기

### 첫 실행
1. 게임을 실행하면 환영 화면이 나타납니다.
2. 이름을 입력하고 "시작하기" 버튼을 클릭합니다.
3. 홈 화면으로 이동합니다.

### 홈 화면 둘러보기
홈 화면에서는 다음 정보를 확인할 수 있습니다:
- **캐릭터 정보**: 현재 레벨과 경험치
- **보유 코인**: 아이템 구매에 사용할 수 있는 코인
- **정답 수**: 지금까지 맞춘 문제 수
- **연속 학습일**: 매일 연속으로 학습한 일수

## 게임 모드

### 연습 모드 🎯

가장 기본적인 학습 모드입니다.

#### 설정 방법
1. 홈 화면에서 "연습 모드" 선택
2. **연산 타입** 선택:
   - ✖️ 곱셈
   - ➗ 나눗셈
3. **난이도** 선택:
   - 😊 쉬움: 작은 숫자로 시작
   - 🤔 보통: 중간 난이도
   - 😤 어려움: 큰 숫자와 복잡한 계산
4. **문제 수** 선택: 5개~20개 (5개 단위)
5. "시작하기" 버튼 클릭

#### 게임 플레이
1. **문제 읽기**: 화면에 표시된 문제를 읽습니다.
2. **답 입력**: 숫자 키패드로 답을 입력합니다.
3. **나머지 입력** (나눗셈): 나머지가 있는 경우 몫과 나머지를 모두 입력합니다.
4. **확인**: "확인 ✓" 버튼을 클릭합니다.
5. **피드백**: 정답/오답 여부와 함께 격려 메시지가 나타납니다.

#### 도움 기능
- **그림 보기 👁️**: 문제를 시각적으로 표현합니다.
  - 블록, 과일, 동물 등으로 수를 나타냅니다.
  - 곱셈: 같은 수의 묶음을 보여줍니다.
  - 나눗셈: 전체를 여러 그룹으로 나누는 것을 보여줍니다.

- **힌트 💡**: 단계별 힌트를 제공합니다.
  - 첫 번째 힌트: 기본적인 계산 방법
  - 두 번째 힌트: 구체적인 계산 과정
  - 세 번째 힌트: 구구단이나 관련 개념

## 학습 기능

### 학습 대시보드 📊

자신의 학습 기록을 확인할 수 있습니다.

#### 전체 통계
- 총 문제 수
- 정답 수
- 전체 정답률
- 연속 학습일

#### 연산별 통계
**곱셈 통계** (세 자리 수 범위)
- 총 문제 수와 정답 수
- 정답률
- 평균 풀이 시간
- 난이도별 정답률 (쉬움/보통/어려움)

**나눗셈 통계** (세 자리 수 범위)
- 곱셈과 동일한 형식의 통계

#### 학습 팁
대시보드 하단에는 현재 실력에 맞는 학습 팁이 표시됩니다:
- 정답률이 낮으면: 시각적 도움 활용 권장
- 풀이 시간이 길면: 구구단 암기 권장
- 연속 학습일이 짧으면: 매일 학습 권장
- 특정 연산이 약하면: 해당 연산 집중 연습 권장

### 오답 노트 📔

틀린 문제를 모아서 다시 풀 수 있습니다.

#### 기능
- **모든 오답 다시 풀기**: 저장된 모든 오답을 한 번에 풀기
- **개별 다시 풀기**: 특정 문제만 선택해서 풀기
- **삭제**: 이미 이해한 문제는 목록에서 제거
- **정답 확인**: 각 문제의 정답과 풀이 힌트 확인

#### 활용 방법
1. 게임 중 틀린 문제는 자동으로 오답 노트에 저장됩니다.
2. 홈 화면에서 "오답 노트" 선택
3. 틀린 문제를 확인하고 다시 풀어봅니다.
4. 완전히 이해한 문제는 삭제합니다.

## 보상 시스템

### 코인 획득 🪙

코인은 다음과 같이 획득할 수 있습니다:
- **정답**: 문제 1개당 10코인
- **별 보너스**: 
  - ⭐ (1개): 50코인 (정답률 50% 이상)
  - ⭐⭐ (2개): 100코인 (정답률 70% 이상)
  - ⭐⭐⭐ (3개): 150코인 (정답률 90% 이상)
- **업적 달성**: 업적당 100코인

### 경험치와 레벨업 ✨

- **경험치 획득**: 정답 1개당 5 EXP
- **레벨업**: 100 EXP마다 레벨 1 상승
- 레벨이 오르면 홈 화면에서 진행 바가 채워집니다.

### 아이템 상점 🛒

코인으로 캐릭터를 꾸밀 수 있는 아이템을 구매합니다.

#### 아이템 카테고리
1. **모자 🎩**
   - 왕관 👑: 100코인
   - 마법사 모자 🎩: 150코인
   - 졸업 모자 🎓: 200코인
   - 파티 모자 🎉: 120코인

2. **의상 👕**
   - 슈퍼히어로 망토 🦸: 250코인
   - 우주복 🚀: 300코인
   - 닌자 복장 🥷: 280코인

3. **악세서리 ✨**
   - 별 지팡이 ⭐: 180코인
   - 마법 책 📚: 160코인
   - 트로피 🏆: 220코인

4. **배경 🖼️**
   - 우주 배경 🌌: 350코인
   - 무지개 배경 🌈: 300코인
   - 성 배경 🏰: 400코인

#### 아이템 사용
1. 아이템 구매 후 "착용하기" 버튼 클릭
2. 같은 종류의 아이템은 하나만 착용 가능
3. "착용 중" 표시가 있는 아이템은 현재 착용 중
4. 언제든지 다른 아이템으로 변경 가능

### 업적 시스템 🏆

다양한 업적을 달성하면 보상을 받습니다.

#### 업적 목록
1. **첫 걸음** 🎯
   - 조건: 첫 문제 풀기
   - 보상: 100코인

2. **열심히 공부** 📚
   - 조건: 문제 10개 풀기
   - 보상: 100코인

3. **완벽해요!** ⭐
   - 조건: 10문제 연속 정답
   - 보상: 100코인

4. **꾸준함의 힘** 🔥
   - 조건: 7일 연속 학습
   - 보상: 100코인

5. **곱셈 마스터** ✖️
   - 조건: 곱셈 문제 50개 풀기
   - 보상: 100코인

6. **나눗셈 마스터** ➗
   - 조건: 나눗셈 문제 50개 풀기
   - 보상: 100코인

## 팁과 전략

### 초보자를 위한 팁

1. **쉬운 난이도부터 시작하세요**
   - 처음에는 쉬운 난이도로 자신감을 키우세요.
   - 정답률이 90% 이상이 되면 다음 난이도로 도전하세요.

2. **시각적 도움을 활용하세요**
   - 그림으로 보면 이해가 쉬워집니다.
   - 특히 나눗셈은 그림으로 보는 것이 도움이 됩니다.

3. **힌트를 적극 활용하세요**
   - 막히면 힌트를 보는 것이 좋습니다.
   - 힌트를 보고 이해한 후 비슷한 문제를 더 풀어보세요.

4. **매일 조금씩 연습하세요**
   - 하루에 10문제씩만 풀어도 실력이 늘어납니다.
   - 연속 학습일을 늘려보세요!

### 중급자를 위한 팁

1. **시간을 재면서 풀어보세요**
   - 정답률이 높아지면 속도도 중요합니다.
   - 평균 시간을 줄이는 것을 목표로 하세요.

2. **오답 노트를 활용하세요**
   - 틀린 문제는 반복해서 풀어보세요.
   - 같은 유형의 문제를 여러 번 풀면 실력이 늘어납니다.

3. **다양한 난이도에 도전하세요**
   - 쉬운 문제만 풀지 말고 어려운 문제도 도전하세요.
   - 어려운 문제를 풀면 더 많은 경험치를 얻을 수 있습니다.

4. **구구단을 완벽하게 외우세요**
   - 구구단을 외우면 곱셈과 나눗셈이 훨씬 쉬워집니다.
   - 2단부터 9단까지 완벽하게 외워보세요.

### 고급자를 위한 팁

1. **암산 연습을 하세요**
   - 힌트 없이 빠르게 풀어보세요.
   - 큰 숫자의 곱셈과 나눗셈에 도전하세요.

2. **모든 업적을 달성하세요**
   - 업적을 모두 달성하면 실력이 크게 향상됩니다.
   - 특히 "완벽해요!" 업적은 집중력을 키워줍니다.

3. **어려운 난이도를 마스터하세요**
   - 어려운 난이도에서 90% 이상의 정답률을 목표로 하세요.
   - 두 자리 수 곱셈과 나눗셈을 완벽하게 익히세요.

4. **친구들과 경쟁하세요**
   - 친구들과 점수를 비교해보세요.
   - 서로 격려하며 함께 실력을 키워보세요.

## 자주 묻는 질문 (FAQ)

### Q: 데이터가 저장되나요?
A: 네! 모든 학습 기록은 브라우저에 자동으로 저장됩니다. 같은 브라우저에서 다시 접속하면 이어서 할 수 있습니다.

### Q: 코인을 더 빨리 모으려면?
A: 정답률을 높여서 별을 많이 받고, 업적을 달성하세요. 특히 3개 별을 받으면 보너스 코인이 많습니다.

### Q: 레벨은 최대 몇까지인가요?
A: 레벨에는 제한이 없습니다! 계속 문제를 풀면 레벨이 계속 올라갑니다.

### Q: 오답 노트에서 문제를 삭제하면 어떻게 되나요?
A: 삭제한 문제는 오답 노트에서 사라지지만, 통계에는 영향을 주지 않습니다.

### Q: 난이도는 언제 올려야 하나요?
A: 현재 난이도에서 정답률이 90% 이상이고, 평균 시간이 15초 이하가 되면 다음 난이도로 올리는 것이 좋습니다.

### Q: 나눗셈에서 나머지는 어떻게 입력하나요?
A: 나머지가 있는 문제는 몫과 나머지를 모두 입력해야 합니다. 화면에 "몫"과 "나머지" 입력란이 따로 표시됩니다.

## 학부모님을 위한 가이드

### 학습 효과
이 게임은 다음과 같은 학습 효과가 있습니다:
- **개념 이해**: 시각적 표현으로 곱셈과 나눗셈의 의미를 이해
- **계산 능력**: 반복 연습으로 계산 속도와 정확도 향상
- **학습 동기**: 게임 요소로 자발적인 학습 동기 부여
- **자신감**: 성공 경험을 통한 수학 자신감 향상

### 권장 사용 시간
- **하루 15~20분**: 집중력을 유지할 수 있는 적절한 시간
- **주 5일 이상**: 꾸준한 학습이 중요합니다
- **휴식**: 30분 이상 연속 사용은 피해주세요

### 학습 진도 확인
- **대시보드**: 자녀의 학습 통계를 확인하세요
- **정답률**: 70% 이상이면 적절한 난이도입니다
- **연속 학습일**: 매일 학습하는 습관을 격려해주세요
- **오답 노트**: 어려워하는 유형을 파악하고 도와주세요

### 칭찬과 격려
- 결과보다는 노력을 칭찬해주세요
- 틀린 문제도 배움의 기회임을 알려주세요
- 작은 성취도 함께 기뻐해주세요
- 비교보다는 개인의 성장에 초점을 맞춰주세요
